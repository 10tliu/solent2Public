Train Ticket System Demo

This demo allows you to simulate a train ticket system.
You can run the demo in two modes depending on which link you select below.
This simulates whether the user is logged in as an administrator or is simply a customer.
As an administrator you can add destinations and timetabled journeys with prices.
As a customer, you can only select a journey to a destination and purchase a ticket.
